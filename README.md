\# Jarvis Local



A local AI assistant built in Python.



\## Features



\- Offline LLM using Ollama

\- Local RAG

\- Semantic document search

\- ChromaDB vector database

\- PDF support

\- Completely offline



\## Requirements



\- Python 3.11

\- Ollama



Install dependencies:



```bash

pip install -r requirements.txt

```



Download the model:



```bash

ollama pull llama3.1:8b

```



Run:



```bash

python main.py

```
